# mo-nvim-override

Replaces `vim` with `nvim`. No-op if `nvim` is not installed.

| Command | Description |
|---------|-------------|
| `vim` | opens `nvim` |

To bypass: use `\vim` (backslash-quoting skips aliases in any shell).

**Dependencies:** `nvim` — falls back to system `vim` if not installed.
